Review Mobile Legend





Mobile Legend adalah sebuah game moba 5v5 yang bisa dimainkan di Mobile Android atau iOS, mobile legend sendiri sangat populer di kalangan anak muda Indonesia terutama para bocah kimcil barbar. Game ini sudah di download lebih dari 100 juta di Playstore dan 50 juta di App Store, menjadikan game ini sangat populer


Menurut saya game ini sangatlah seru jika dimainkan bersama teman-teman, akan tetapi, saya tidak menganjurkan bermain Ranked sendiri (Solo Player) karena setelah saya meriset sendiri selama berbulan bulan, jika kamu sampai nekad bermain Mobile Legend sendiri (Solo Player) akan mengeluarkan sifat buruk dan perkataan kotor dari mulut anda sendiri(SANGAT TIDAK DI ANJURKAN!). Game ini pun mempunyai range power antar hero yang lumayan besar, hal ini di karenakan marketing sales dari Moonton (developer game) yang sering mengeluarkan patch untuk menyesuaikan power supaya hero baru yang akan datang atau yang baru datang terasa over power sehingga orang akan membeli hero tersebut.


Lain halnya jika kalian bermain 3 atau 5 Player sekaligus atau biasa disebut "Party", game ini akan terasa sangat menyenangkan dan seru karena akan mengisi waktu-waktu anda


Kesimpulannya adalah Game ini sangatlah populer, semua kalangan memainkan game ini, dari satpam hingga penjabat daerah atau mungkin bahkan presiden sendiripun memainkan game ini, kendati game ini mempunyai sisi negatif lumayan banyak
